# Глоссарий: заказ на выдачу через Telegram

## Граница

Один glossary-объект: термины бизнес-контекста `Expressa v1` для заказа на выдачу через Telegram.

## Термины

| Термин                            | Определение                                                                                                                                                         |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Customer                          | Клиент кафетерия, оформляющий заказ на выдачу через клиентского бота и клиентский веб-интерфейс.                                                                    |
| Barista                           | Сотрудник, который обрабатывает заказы в backoffice: подтверждает, отклоняет, отмечает готовность, закрывает заказ после выдачи и временно меняет доступность меню. |
| Administrator                     | Пользователь backoffice с расширенными правами на управление меню, слотами, ролями пользователей, блокировкой и разблокировкой пользователей.                       |
| Главный administrator             | Administrator, который существует как исходный носитель административных прав при запуске продукта и может назначать других administrator и barista.                |
| Управление доступом пользователей | Бизнес-сценарий экрана `Пользователи`, который объединяет назначение ролей, блокировку и разблокировку пользователей.                                               |
| Клиентский бот                    | Telegram-бот, через который customer открывает клиентский веб-интерфейс и получает уведомления по заказу.                                                           |
| Служебный бот                     | Отдельный Telegram-бот, через который barista и administrator открывают backoffice и получают служебные уведомления.                                                |
| Backoffice                        | Единый веб-интерфейс для barista и administrator с разграничением доступа по ролям.                                                                                 |
| Заказ на выдачу                   | Заказ, который customer оформляет заранее и получает офлайн при выдаче в заведении.                                                                                 |
| Категория меню                    | Группа товаров, внутри которой customer просматривает меню.                                                                                                         |
| Товар                             | Позиция меню, которую customer может добавить в корзину.                                                                                                            |
| Напиток                           | Товар, для которого выбор размера `S`, `M` или `L` обязателен и влияет на цену.                                                                                     |
| Группа дополнительных опций       | Набор дополнительных опций, назначаемый на группу меню и распространяющийся на все товары этой группы.                                                              |
| Дополнительная опция              | Дополнение к товару, которое может быть платным или бесплатным; в пределах группы может быть взаимоисключаемым с другими опциями.                                   |
| Корзина                           | Набор выбранных customer товаров и опций до создания заказа.                                                                                                        |
| Слот выдачи                       | Интервал времени внутри рабочих часов, доступный для выбора при оформлении заказа.                                                                                  |
| Активный заказ                    | Заказ в статусе `Создан`, `Подтвержден` или `Готов к выдаче`, занимающий вместимость слота.                                                                         |
| История заказов                   | Список заказов customer, доступный для просмотра после их создания.                                                                                                 |
| Заблокированный пользователь      | Пользователь, которому administrator запретил использование приложения.                                                                                             |
| Разблокированный пользователь     | Пользователь, которому administrator восстановил доступ к приложению в соответствии с сохраненными ролями.                                                          |
