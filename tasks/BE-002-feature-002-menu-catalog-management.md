# Карточка задачи

## Карточка задачи

- Идентификатор: `BE-002`
- Родительская задача: `FEATURE-002`
- Заголовок: `Серверная часть управления каталогом меню, ценами и группами дополнительных опций`
- Описание: `Нужно реализовать в apps/server серверную часть FEATURE-002: чтение и сохранение каталога меню для administrator, модель категорий и товаров, ценовые правила напитков по размерам S/M/L, группы дополнительных опций с платными, бесплатными и взаимоисключающими вариантами, а также бизнес-ошибки по нарушению правил модели. Если публичный HTTP-контракт потребует новых DTO, они должны быть сначала зафиксированы в packages/shared-types и только затем реализованы в серверной части. Результат должен покрывать только серверные операции управления каталогом и не захватывать клиентский рендеринг вкладки menu.`
- Единица поставки: `FEATURE-002`
- Роль: `Бэкенд`
- Изменяемый контур: `backend`
- Приоритет: `Критический`
- Статус: `Готова к работе`
