# Карта документации `docs/architecture`

Этот каталог хранит архитектурные решения и навигацию по будущему и текущему коду. Источником правды для бизнес-смысла остаются `docs/business/`, для системного поведения — `docs/system/`.

## Как искать документ

- Если нужно понять утвержденный стек и базовые технологические ограничения: `stack.md`.
- Если нужно понять coding standard, правила unit tests и ожидания к ревью: `code-style.md`.
- Если задача относится к первой административной поставке `DU-01`: `du-01-administration.md`.
- Если нужно быстро найти модуль, entrypoint, зависимость, способ запуска или точку обновления документации: `application-map.md`.
- Если задача касается окружений, CI/CD, env vars, smoke-check или rollback: `deployment-map.md`.
- Если нужно сверить уже реализованный backend-срез `DU-01` с целевой архитектурой: сначала `application-map.md`, затем `deployment-map.md`.

## Состав артефактов

- `stack.md`
  - Канонический источник по технологическому стеку и базовой структуре монорепозитория.

- `code-style.md`
  - Общие правила кодстайла, unit tests и review gates для `AR/FE/BE/DO`.

- `du-01-administration.md`
  - Архитектурная рамка первой административной delivery unit: обязательные контуры, исключения из scope и правила декомпозиции child-задач.

- `application-map.md`
  - Обязательная карта приложения: фактические и плановые контуры, модули, entrypoints, зависимости, запуск, тестирование и deployment-маршруты.

- `deployment-map.md`
  - Карта окружений, исполняемых локальных/CI-команд, env vars, smoke-check и rollback.

## Правила актуализации

- Обновляйте `application-map.md`, если меняются структура каталогов, entrypoints, интерфейсы между модулями, shared packages, env/config, способ запуска, тестирования или деплоя.
- Обновляйте `deployment-map.md`, если меняются окружения, pipeline stages, secrets, rollback path или smoke-check.
- Обновляйте этот файл, если в `docs/architecture/` появляется новый обязательный артефакт или меняется маршрут навигации.
- Если изменения делают корневой `README.md` неполным, обновляйте и его.
