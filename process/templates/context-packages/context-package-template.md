# Шаблон контекстного пакета подзадачи

## Назначение

Контекстный пакет используется для передачи одной подзадачи агенту-исполнителю без повторного широкого сбора контекста.

Файл создается во временной рабочей папке по схеме `.agent-work/<TASK-ID>/context-<NN>-<slug>.md`.
