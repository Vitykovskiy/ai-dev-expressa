# AGENTS

## Старт работы

Перед началом любой работы в этом репозитории агент должен сначала прочитать [README.md](./README.md).

## Зачем это нужно

- `README.md` содержит актуальную структуру проекта.
- `README.md` указывает, где находятся основные документы, шаблоны и задачи.
- `README.md` задает базовую точку входа для навигации по репозиторию.

## Обязательное правило

Если агент еще не прочитал `README.md`, он не должен переходить к анализу, изменению файлов или постановке задач.

Перед внесением любых изменений агент должен убедиться, что локальный репозиторий синхронизирован с удаленным и все актуальные изменения с сервера подтянуты, ветка растет от основной ветки. Пока это не проверено, агент не должен приступать к редактированию файлов.

При создании коммитов агент должен использовать формат Conventional Commits.
