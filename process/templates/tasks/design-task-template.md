# Шаблон дизайнерской задачи

## Назначение

Шаблон используется для `DESIGN-*` задач, которые передаются дизайнеру или инструменту подготовки макета как самодостаточный brief.

Разделы для дизайнера описывают только видимые изменения макета, состояния, варианты, тексты и проверки. Аналитическое обоснование, границы фичи, design readiness, handoff и task-tree контекст фиксируются в системных артефактах, а не в brief для дизайнера.

## Ограничения

- Дизайнер меняет только named approved UI source или design artifact.
- Дизайнер не проектирует backend/API.
- Дизайнер не редактирует project docs, process docs, production code, тесты или task tree.
- Вся необходимая постановка содержится в этой карточке.
